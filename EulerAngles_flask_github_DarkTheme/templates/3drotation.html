3D Rotation

When an object is rotated in space, there are different ways to represent the rotation. One representation is through a three-by-three rotation matrix. See the tab below on rotation matrices for more information about this specific representation. A matrix can express the way an object is rotated about three axes, which leads to another type of representation of 3D rotation – Euler Angles. Each axis is associated with an angle that describes the object’s rotation. While Euler Angles can appear to be the easiest way to represent the rotation of an object in 3D space, there is a problem that arises with specific rotations called “Gimbal Lock,” that limits the use of Euler Angles. For this reason, it can be helpful to describe 3D rotation in another way, which is where quaternions come into use. A quaternion can express a rotation when its norm, \(|q|=\sqrt{q_{0}^2+q_{1}^2+q_{2}^2+q_{3}^2}\) is equal to \(1\). In order to find the coordinate point, Y, of an object after it has been rotated from its previous point, X, the following equation is given: \(q_{Y}=q_{A}q_{X}q_{A}^\star\). In this equation, \(q_{A}\) is the rotation, \(q_{A}^\star\) is its conjugate, and both \(q_{X}\) and \(q_{Y}\) are vectors expressed as quaternions with no real part.
