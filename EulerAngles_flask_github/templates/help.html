Quaternions

Quaternions are a 4-dimensional number system such that each quaternion consists of a scalar component and an imaginary vector component. They are an extension of complex numbers in that the vector components \((i, j, k)\) follow the equation \(i^2=j^2=k^2=ijk=-1\) and each of the three units of the vector component represent an imaginary dimension. Putting this vector with the scalar component gives a four number representation for a quaternion, \(q\), where \(q=q_{0}+iq_{1}+jq_{2}+kq_{3}\). While the number system represented by quaternions is not commonly used, it is particularly helpful in describing 3D rotation. More information is given below about its use in 3D rotation.

3D Rotation

When an object is rotated in space, there are different ways to represent the rotation. One representation is through a three-by-three rotation matrix. See the tab below on rotation matrices for more information about this specific representation. A matrix can express the way an object is rotated about three axes, which leads to another type of representation of 3D rotation – Euler Angles. Each axis is associated with an angle that describes the object’s rotation. While Euler Angles can appear to be the easiest way to represent the rotation of an object in 3D space, there is a problem that arises with specific rotations called “Gimbal Lock,” that limits the use of Euler Angles. For this reason, it can be helpful to describe 3D rotation in another way, which is where quaternions come into use. A quaternion can express a rotation when its norm, \(|q|=\sqrt{q_{0}^2+q_{1}^2+q_{2}^2+q_{3}^2}\) is equal to \(1\). In order to find the coordinate point, Y, of an object after it has been rotated from its previous point, X, the following equation is given: \(q_{Y}=q_{A}q_{X}q_{A}^\star\). In this equation, \(q_{A}\) is the rotation, \(q_{A}^\star\) is its conjugate, and both \(q_{X}\) and \(q_{Y}\) are vectors expressed as quaternions with no real part. Euler Angles and rotation matrices both can be converted to quaternions. However, these conversion methods come with their own set of challenges which is described in further detail below.

Rotation Matrices to Quaternions

Given a rotation matrix, \(A\), the following equations show the methods for obtaining the corresponding quaternion: \[|q_{0}|=\sqrt{\frac{\text{Trace}(A)+1}{4}}\] \[|q_{1}|=\sqrt{\frac{A_{11}}{2}+\frac{1-\text{Trace}(A)}{4}}\] \[|q_{2}|=\sqrt{\frac{A_{22}}{2}+\frac{1-\text{Trace}(A)}{4}}\] \[|q_{3}|=\sqrt{\frac{A_{33}}{2}+\frac{1-\text{Trace}(A)}{4}}\] where \(\text{Trace}(A)=A_{11}+A_{22}+A_{33}\).

Euler Angles to a Quaternion

Based on the conversions shown above, Euler Angles to a rotation matrix and a rotation matrix to a quaternion, it is relatively simple to piece together how one might convert Euler angles to a quaternion. Since Euler angles have twelve different rotation sequences, it is important to note that once the matrices corresponding to the rotations around each axis are constructed, they must be multiplied in the correct order for the desired sequence.

Though it might be worthwhile to construct the conversion methods of Euler angles to a quaternion using the information above, the simplified conversion methods are also given below based on each series of rotation.

XYZ Rotation

\[q_{0}=\cos(\frac{\phi}{2}) \cos(\frac{\theta}{2}) \cos(\frac{\psi}{2}) - \sin(\frac{\phi}{2}) \sin(\frac{\theta}{2}) \sin(\frac{\psi}{2})\] \[q_{1}=\sin(\frac{\phi}{2}) \cos(\frac{\theta}{2}) \cos(\frac{\psi}{2}) + \cos(\frac{\phi}{2}) \sin(\frac{\theta}{2}) \sin(\frac{\psi}{2})\] \[q_{2}=\cos(\frac{\phi}{2}) \sin(\frac{\theta}{2}) \cos(\frac{\psi}{2}) - \sin(\frac{\phi}{2}) \cos(\frac{\theta}{2}) \sin(\frac{\psi}{2})\] \[q_{3}=\sin(\frac{\phi}{2}) \sin(\frac{\theta}{2}) \cos(\frac{\psi}{2}) + \cos(\frac{\phi}{2}) \cos(\frac{\theta}{2}) \sin(\frac{\psi}{2})\]

XZY Rotation
