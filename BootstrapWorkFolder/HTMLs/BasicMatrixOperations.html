Matrix to Matrix Multiplication

Matrix multiplication is a bit more complex than addition or subtraction and in order to multiply two matrices, the number of columns in the first matrix must equal the number of rows in the second matrix. The resulting matrix has the same number of rows as the first matrix and the same number of columns as the second matrix.

Properties of Matrix Multiplication

  • Not commutative: For two matrices \(A\) and \(B\), \(A\times B\neq B\times A\)
  • Associative: For three matrices \(A\), \(B\), and \(C\), \(A\times(B\times C)=(A\times B)\times C\)
  • Distributive: For three matrices \(A\), \(B\), and \(C\), \(A\times(B+C)=A\times B + A\times C\)
  • Identity Matrix: An identity is some element in a set that when combined with the other elements of that set in some sort of mathematical transformation, leaves that element unchanged. For example, the number 1 is the identity for the multiplication of all real numbers since any number multiplied by one is equal to itself. In the set of matrices, the identity for multiplication is a matrix with an equal number of rows and columns, all ones in its diagonal, and zeros for every other value as shown below:
    Any matrix multiplied by an identity matrix of the right dimensions will be equal to itself, and so the following formula holds for any matrix \(A\) and an identity matrix \(I\): \[A\times I = I\times A = A\]
